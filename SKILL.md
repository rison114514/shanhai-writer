# 山海写作

## 技能架构

```
shanhai-writer
├── 创作模式 → rui-writer（逐段协作写作）
│   └── 继承 zhang-writer（16条公理/不修正原则/风格内核/禁区）
└── 降AI模式 → de-ai（5步快速路径 + 执行纪律）
```

## 自动路由

| 用户意图 | 触发词 | 路由到 |
|---|---|---|
| 写一篇新文章 | 写文章、写稿子、公众号、帮我写、逐段写 | `rui-writer` 子技能 |
| 降低已有文章的AI率 | 降AI、反AI、去AI味、降低AIGC | `de-ai` 子技能 |

## 使用方式

**创作**：说「写一篇关于 XX 的文章」，自动进入 rui-writer 逐段协作流程。逐段询问意图和素材，AI 只填空不扩写。

**降AI**：发文章说「降AI」，自动进入 de-ai 5步快速路径。4版迭代完成。

## 子技能

- `rui-writer` → 逐段协作写作，人给骨架，AI 填空 + 搜资料
- `de-ai` → 降 AI 率，反 AI 基线 + 样板文参考
- `zhang-writer` → 参考库，16 条公理、不修正原则、风格内核、禁区

## 执行参考

创作时加载 `rui-writer` 子技能，降AI时加载 `de-ai` 子技能。两个子技能均继承 `zhang-writer` 底层规则。
